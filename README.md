## 📌 Overview  

This **Beyond Curriculum Training Program** is designed for **ECE students (2023–27 batch)** focusing on **Embedded Systems and IoT**.  

It bridges the gap between **academic theory** and **industry-level practical implementation**, covering both **hardware and software domains**.

---

## 🎯 Key Highlights  

- 🔧 Hands-on Embedded Programming  
- ⚙️ Microcontroller Architecture (8051)  
- 💻 Embedded C & RTOS Concepts  
- 📡 IoT Communication Protocols  
- ☁️ Cloud Integration  
- 🤖 Real-world Applications

# 📘 Curriculum  

---

## 🔹 Module 1: Embedded Processor Architecture  

🧠 *Core of Embedded Systems*  

- Harvard vs Von Neumann Architecture  
- CISC vs RISC  
- Memory Organization (RAM/ROM)  
- Stack & Interrupts  
- Timers, Counters, UART  
- Parallel Port Programming  

---

## 🔹 Module 2: Embedded C & RTOS  

💻 *Efficient and Real-Time Programming*  

- Embedded C Basics  
- Memory & I/O Interfacing  
- RTOS Concepts:  
  - Task Scheduling  
  - Context Switching  
  - Priority Scheduling  
  - Inter-Process Communication  

---

## 🔹 Module 3: IoT Fundamentals & Arduino  

🌐 *Connecting Physical Devices*  

- IoT Architecture  
- M2M Communication  
- Arduino Programming  
- Sensors: Temperature, IR, LDR  
- Actuators: Motors, Relays  

---

## 🔹 Module 4: Communication Protocols  

📡 *How Devices Communicate*  

- Bluetooth  
- Wi-Fi  
- ZigBee  
- MQTT  
- LoRa  

💻 Platforms:  
- Raspberry Pi  
- GPIO Programming  
- Linux Environment  

---

## 🔹 Module 5: IoT Applications  

🚀 *Real-World Implementation*  

- 🌱 Smart Agriculture  
- 🏠 Home Automation  
- 🌆 Smart Cities  
- ❤️ Healthcare Systems  

☁️ Cloud Integration:  
- Data Logging  
- Remote Monitoring  
- Signal Processing
